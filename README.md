# SeekerDOS



A DOS-like Shell for use with CodeChum and its Playground feature



##### Information

* SeekerDOS is a shell that is used to run software inside CodeChum's Playground.
* SeekerDOS can run any Linux command by typing `/bin/<command>`.
* SeekerDOS can read or copy files.
* SeekerDOS can do anything with the `/bin/sh` command.



##### Usage



###### Requirements

* A CodeChum account with an active subscription
* Access to CodeChum Playground



###### Running the code

* Open a version in the "Releases" section.
* In CodeChum, click on "Playground" (with the `<>` symbol), then set the language to C#.
* Copy everything from the selected version .cs file to the playground.
* Click "Run code".



##### Commands

* `dir`
* `read <filename>`
* `copy <filename> <absolute file path>`
* `cd <folder name>`
* Anything inside the /bin directory (available in `SeekerDOS_main.cs`, will be released in Petuni a2.2)
